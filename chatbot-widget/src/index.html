Chatbot Widget Demo

This page demonstrates the chatbot widget. Look for the blue chat button in the bottom-right corner.

Click it to open the 800px wide chat panel that slides in from the right.

How to Embed

To embed this widget on any website, simply add this script tag to your HTML:

<script src="widget.js"></script>

That's it! The widget will automatically create the chat button and panel.

Features

  • Floating chat button with emoji icon
  • 800px wide slide-out panel
  • Full height container
  • Smooth animations
  • Close button inside the panel
  • No dependencies required
